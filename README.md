## Overview:-
platform for managing and showcasing games with full CRUD operations, validations, and dynamic UI enhancements.


## Key Highlights:-

- Implemented complete CRUD functionality with server/ client-side validations, image upload & preview,
  and device/ category management using EF Core & Repository Pattern.
- Enhanced user experience with AJAX-based actions, and SweetAlert confirmations for smooth and responsive UI interactions.


## Tech Stack:-

ASP.NET Core MVC | EF Core | LINQ | Repository Pattern | SQL Server | Bootstrap
